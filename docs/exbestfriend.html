Dear Ex-Bestfriend

Dear Ex-Bestfriend,
Frankly,I don't miss you now
But I miss the memories which were wow.
I never thought I would write this letter,
For I knew we were always better together.
There were some things upon which we didn't agree
And then came the day we were set free
From always laughing on eachother,
To never talking with onne another.
I always wonder what went wrong with us,
As we never discussed ,
The thing that broke our trust.
However my friend,
I don't miss you now,
But the memories which were wow.

~Pragni Naik
